Discription:

This is my first homework in ADB 2013 course in NTHU. I try to build a Spatial Application using Javascript, djang, and google map api v3 with a Spatial Database named postgresql + postGIS. The function of this application is that we could set a query box on the map and query the information of the cities in the query box.

We can calculate the distance between 2 cities in the query box. We can choose a country in the query box and list the cities in this country. We can select a population class and list the cities whose population is in this class. We can choose a city in the query box and get nearest n cities from the chosen city.

Function:

  1. In "Home" page, we can see many red markers on the map. These red markers mean the locations of the cities on the map. These cities' information are list in the "Information of Total Cities". We can draw a query box on the map by using the google map's draw tool. We choose "draw a rectangle" from the "drawing tool". When we draw a query box on the map, we can see the query box's minimum longitude, minimum latitude, maximum longtitude, and maxmum latitude in the "Query Box Information". If the query box is what we want, we press the "Set Query Box" button. Otherwise, we press the "Remove Query Box" to clear the rectangle on the map.
  2. When we press the "Set Query Box" button, it will generate new tab "Cities in The Query Region" page. We can see blue markers on the map. These blue markers are cities in the query box we just set. These cities' information are list in the "Information of Cities in Query Box".
  3. If we want to calculate distance between 2 cities, we click "Get Distance". It generate a new tab "Get Distance between 2 Cities". We can select from which city to which city and press "Get Distance" button and the result will be given below.
  4. If we want to list cities in the country, we click "Get Cities from The Chosen Country". It generate a new tab "Get Cities from The chosen Country". We can select the target country and press "Search" button and we will get the information below.
  5. If we want to list cities whose population are in the chosen population class, we click "Get Cities By Population". It generate a new tab "Get Cities By population". We can select the population class and press "Search" button and we will get the information below.
  6. If we want to list nearest n cities from the chosen city, we click "Get Nearest n Cities". It generate a new tab "Get Top n Nearest Cities". We can select the target city, enter integer value of n, and press "Search" button and we will get the information below.
